# Configuration

Planned variables:

| Variable | Purpose |
|---|---|
| `CRUX_CONSOLE_API_URL` | Crux Server API URL. |
| `CRUX_CONSOLE_AUTH_MODE` | `none`, `oidc`, or cloud-hosted auth mode. |
| `CRUX_CONSOLE_OTEL_ENABLED` | Enable frontend telemetry. |
| `CRUX_CONSOLE_TRACE_REDACTION` | Redaction level for displayed trace payloads. |
